## How to Add Missing Sprites

1. Generate or find PNG images for the missing monsters
2. Place them in: `client/src/assets/monsters/`
3. Use exact filenames as listed above
4. Recommended size: 512x512px or larger
5. Transparent backgrounds work best

The game will automatically use any sprites that are present!
