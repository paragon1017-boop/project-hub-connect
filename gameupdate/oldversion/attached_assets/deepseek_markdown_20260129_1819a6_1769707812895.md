# WebGL Dungeon Crawler

A WebGL-based dungeon crawler game with TypeScript.

## Installation
1. Install Node.js (v16 or higher)
2. Run: `npm install`
3. Build: `npm run build`
4. Run: `npm start`
5. Open: http://localhost:8080

## Controls
- **WASD** or **Arrow Keys**: Move
- **Space**: Attack/Interact
- **I**: Inventory

## Features
- WebGL 2.0 rendering
- Procedural dungeon generation
- Equipment system with enhancement
- Combat system with abilities
- Multiple character classes
- Progressive difficulty floors
